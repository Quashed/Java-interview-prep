# Core Java 
---
## Data Types

### Primitive types: 
The actual values are stored in memory.
- integer: byte (8), short (16), int (32), double (64)
- decimal: float (32), double (64)
- character: char (16)
- logical: boolean (1)

### Reference types:
References pointing to objects are stored in memory.
- String
- Arrays
- Classes/objects
- Interfaces
